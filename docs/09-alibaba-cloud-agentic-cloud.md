## Two Pillars: AI Native Cloud + Agent Native Cloud

### AI Native Cloud
> Supports model training & token production

#### Training
- Large-scale pre-training
- Diverse fine-tuning strategies
- End-to-end ALVR support
- Compatibility with heterogeneous chips

#### Inference
- High stability
- Inference network optimization
- Multi-level KV cache
- Multi-strategy parallelism
- Fine-grained scheduling
- Intelligent routing & load balancing

### Agent Native Cloud
> Hosts massive Agent applications, called by agents

#### Agent Infrastructure
- Runtime
- Orchestration
- Memory
- Agent OS
- Observability

#### Agentic Products
- Cloud services tailored to Agents
- Supports Agentic workloads

### Data Plane
- Collection & Ingestion
- Storage & Governance
- Data Semantics
- Multi-level Memory
- Context Assembly
